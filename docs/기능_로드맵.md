# 경비 정산 시스템 — 기능 로드맵

**문서 버전**: v1.0
**작성일**: 2026-04-15
**작성**: AX 혁신실

> 현재 v26.04 기준으로 이후 추가 예정/검토 기능 정리.

---

## 🔥 우선 (단기 · 업무 임팩트 큼)

| # | 기능 | 설명 | 상태 |
|---|---|---|---|
| 1 | **엑셀(XLSX) 내보내기** | 회계팀이 ERP에 업로드하기 위한 구조화 엑셀. 비고 태그 파싱해서 거래처·참석자 열 분리 | 🚧 진행 |
| 2 | **원본 영수증 PDF 병합** | 정산서 인쇄 시 업로드한 영수증 원본들을 자동으로 뒤에 첨부 (증빙 일체) | 🚧 진행 |
| 3 | **회계팀 검증 툴 연동** | 제출 정산서 자동 검증 (식비 한도 · 카테고리 · 합계 검산). 별도 기획서 참조 | 🚧 진행 |
| 4 | **이메일 전송 버튼** | 완성된 정산서 PDF + 첨부 영수증을 회계팀 메일로 원클릭 전송 | 대기 |
| 5 | **개인 프로필 저장** | 성명·부서·차량 정보 등을 브라우저에 기억, 다음 정산 시 자동 채움 | 🚧 진행 |

---

## 🌱 중기 (편의성)

| # | 기능 | 설명 |
|---|---|---|
| 6 | **카드 명세서 CSV 자동 import** | 법인카드 명세서 업로드 → 카테고리 자동 분류 → 행 일괄 추가 |
| 7 | **공휴일 API 연동** | 공공데이터포털 특일 API → 유가 조회·출장일 계산 시 공휴일 정확히 제외 |
| 8 | **월별 사용 이력 대시보드** | 본인 월별 경비 누계, 한도 대비 사용률, 카테고리별 트렌드 그래프 |
| 9 | **영수증 중복 강탐지** | 같은 금액·날짜·업체 이미 제출됐는지 DB 체크 (서버 저장 필요) |
| 10 | **필수 필드 제출 전 검증** | 작성자 미입력 / 증빙 없음 / 적격증빙 누락 등 제출 전 일괄 체크리스트 |
| 11 | **빠른 입력 단축키** | Tab으로 다음 셀, Enter로 행 추가, Ctrl+D로 중복 등 |

## 🚀 장기 (전략)

| # | 기능 | 설명 |
|---|---|---|
| 12 | **모바일 PWA / 카메라 촬영** | 폰으로 영수증 즉시 촬영 → 자동 업로드. 출장 중 실시간 입력 |
| 13 | **AI 학습/교정 피드백 루프** | 사용자 수정 패턴을 기록해 AI 정확도 점진 개선 |
| 14 | **예산 사전 시뮬레이션** | 제출 전 "이 건 접대비 예산 80% 초과" 같은 사전 경고 |
| 15 | **QR 영수증 수집함** | 업체에서 QR 스캔 → 공용 업로드 URL로 바로 입력 (공동 출장 시) |
| 16 | **결재 워크플로우** | 부서장 승인 단계 · 반려 사유 기록 · 전자서명 |
| 17 | **ERP 직접 연동** | 회계 시스템으로 전표 자동 생성 (SAP/더존 등) |
