1. 시작하기

본 시스템은 회사 개인경비 영수증을 업로드하면 AI가 자동으로 금액·날짜·카테고리를 인식하여 정산서를 작성해 주는 도구입니다.

기본 흐름

  1. 상단 탭에서 국내 개인경비 또는 해외 개인경비 선택
  2. 작성자 정보(성명 · 소속 부서 · 정산 차수 · 작성 일자) 입력
  3. 영수증 업로드(AI 분석) 또는 항목 추가 버튼으로 수동 입력
  4. 필요 시 차량 유류비 · 출장일비 입력
  5. 정산서 인쇄 버튼으로 출력 또는 PDF 저장

2. 영수증 AI 자동 분석

업로드 방법

  • 업로드 영역을 클릭하여 파일 선택 (여러 장 가능)
  • 파일을 영역으로 드래그&드롭
  • 클립보드에 캡처한 이미지를 Ctrl + V 붙여넣기
  • 지원 포맷: 이미지(PNG · JPG 등) · PDF
✅ 기존 입력 내역은 유지됩니다. 영수증을 추가로 업로드해도 이미 입력한 행은 그대로 남고, 새 영수증만 하단에 추가됩니다. (중복 업로드 시 같은 영수증은 자동 무시)

카테고리 정확도 개선 (3단계)

A. 프롬프트 기반 상점 매핑 — 다이소·모던하우스·이케아 등 잡화점은 소모품으로, 스타벅스·이디야 등 카페는 식비로 자동 확정
B. 학습 — AI 분류를 사용자가 수정하면(다른 카테고리 칸으로 옮기면) 해당 상점은 다음부터 올바르게 분류됨 (서버 저장)
C. 파일명 규칙 — 파일명에 키워드 포함 시 강제 분류
· 접대_... → 접대비
· 주유_..., 유류_... → 유류비
· 택시_..., 교통_... → 교통비
· 주차_... → 주차비
· 숙박_..., 호텔_... → 숙박비
· 소모품_..., 다이소_... → 소모품
· 택배_..., 운반_... → 운반비
⚠ 한 파일당 영수증 1장만 업로드하세요.
여러 영수증이 한 파일 안에 같이 스캔·촬영되어 있으면 AI가 가장 눈에 띄는 한 건만 인식하거나, 금액·항목을 뒤섞어 잘못 판독할 수 있습니다.
· PDF 여러 페이지 = 각 페이지가 별도 영수증으로 처리됨 (OK)
· 이미지 한 장에 영수증 2장 붙여넣기 = 권장하지 않음 (1장당 1파일 원칙)

2-1. 파일명 규칙 — 자동 분류·인원·숙박일수

영수증 업로드 시 파일명에 다음 키워드를 포함하면 AI 분석보다 우선하여 자동 적용됩니다. 파일명과 영수증 내용이 다를 때 유용합니다.

인원수 · 숙박일수 자동 인식

  • 파일명에 3명, 3인, 3인분 → 인원수 3명 자동 반영
  • 파일명에 2박 → 숙박일수 2박 자동 반영
  • "일"은 날짜(3일, 15일)와 혼동되므로 숙박일수엔 "박"만 인식
  • 인식되지 않으면 기본 1명 / 1박 처리, 비고 칸에 [3명] 또는 [2박] 직접 적어도 동일하게 반영

파일명 예시

· 접대_ABC상사_저녁.jpg → 접대비 + AI가 상세 파싱
· 점심_3명.jpg → 식비, 인원 3명 (한도 36,000원 적용)
· 호텔_2박.jpg → 숙박비, 2박 (한도 140,000원 적용)
· 주유_경부선.pdf → 유류비
· 다이소_사무용품.jpg → 소모품 (AI가 잘못 분류해도 강제)
· 택시_출장이동.jpg → 교통비
💡 활용 팁 — AI가 자주 잘못 분류하는 영수증은 파일명에 키워드를 박아서 업로드하면 가장 확실합니다. 학습된 상점이 많아질수록 수작업도 점점 줄어듭니다.

2-2. 행별 영수증 미리보기 · 수동 첨부 (📎)

모든 행 액션 셀에 📎 페이퍼클립 아이콘이 두 가지 상태로 동작합니다.

두 가지 상태 (자동 전환)

상태아이콘동작
영수증 없음 (수기 입력 행 또는 AI 인식 실패)📎+ 회색클릭 → 파일 선택 → 영수증 첨부 후 자동으로 미리보기 상태로 전환
영수증 있음 (AI 분석 또는 수동 첨부)📎 파란색클릭 → 모달에서 원본 영수증 즉시 표시

지원 포맷 (미리보기)

  • 이미지: PNG/JPG/GIF/WEBP/BMP → 모달 내 직접 표시
  • PDF → 인라인 뷰어
  • 그 외 → 다운로드 링크
  • 모달 닫기: 닫기 버튼 / × / 모달 바깥 클릭

수동 첨부 활용 (AI 인식 실패 또는 수기 행)

AI가 영수증을 인식하지 못하거나, 수기로 항목을 추가했을 때:
1. 해당 행의 회색 📎+ 아이콘 클릭 → 파일 선택 다이얼로그
2. 영수증 이미지 또는 PDF 선택
3. 자동으로 파란 📎 미리보기 아이콘으로 전환되고 증빙 PDF 병합 시 자동 포함
4. 같은 파일명이 이미 있으면 _1, _2 접미사로 충돌 회피
⚠ 새로고침/저장본 로드 시 한계 — 영수증 원본 파일은 브라우저 메모리에만 보관됩니다. 페이지 새로고침 또는 저장본 불러오기 후에는 행은 그대로 남지만 미리보기·증빙PDF 포함·인쇄 첨부가 안 됩니다 (영수증 재첨부 필요).
💡 영수증 첨부의 효과 (3가지)
1. 행별 📎 미리보기 가능
2. 증빙 PDF 병합 시 자동 포함
3. 정산서 인쇄 시 마지막 페이지 다음에 영수증 페이지가 자동 첨부되어 함께 출력됨

2-3. 연도 컬럼 · 날짜순 정렬

국내·해외 정산 테이블 첫 번째 컬럼이 연(年) 입니다. AI가 영수증 날짜를 인식하면 자동으로 채워지고, 수기 입력 행은 작성일자의 연도가 기본값으로 들어갑니다.

자동 정렬 규칙

  • 연 → 월 → 일 순으로 자동 정렬
  • 연도가 다른 영수증이 섞여 있어도 (예: 12월 영수증을 1월에 정산 시) 정확한 시간 순으로 정렬됨
  • 연 칸을 직접 수정하면 즉시 재정렬
저장본을 불러올 때도 연도가 함께 복원됩니다. 이전 버전(연도 컬럼 없는 저장본)은 작성일자 연도로 자동 채워집니다.

2-4. AI / 수기 입력 행 구분 배지

각 행의 비고 셀 좌측 상단에 작은 색상 배지가 표시되어 영수증 출처를 한눈에 식별할 수 있습니다.

출처화면 배지 (비고 셀 좌측)인쇄 시 (비고 셀 우측)
AI 분석 영수증 (자동 인식) AI (인디고) 표시 안 됨 (자동 분석 = 기본 신뢰)
수기 입력 행 (직접 입력 / 유류비 정산 반영) 수기 (회색) 수기 (흰 바탕 검정 테두리/글씨)
왜 인쇄에서 수기 배지만 유지하는가
AI 분석 영수증은 원본 이미지가 첨부되어 객관적이지만, 수기 입력 행은 사용자가 직접 작성한 항목이라 감사 시 별도 검토가 필요합니다. 인쇄 출력물에 "수기" 배지를 남겨두면 회계팀이 어느 행이 수기 입력인지 즉시 구분할 수 있습니다.
화면에서는 비고 셀 좌측에 큰 배지로, 인쇄에서는 우측(영수증 미리보기 아이콘 자리)에 작은 흑백 배지로 표시되어 인쇄 흐름을 방해하지 않습니다.
행에 마우스를 올리면 툴팁으로 "AI 분석 영수증" 또는 "수동 입력 행" 표시.
⚠ 수기 입력 행은 영수증 첨부 필수
수기 입력 행이라도 회색 📎+ 버튼으로 영수증을 반드시 첨부해야 합니다. 영수증 미첨부 수기 행이 하나라도 있으면 정산서 인쇄가 차단됩니다 (검증 버튼 결과에도 🔴 오류로 표시).
예외 — 영수증 면제 (rev. 5)
국내 표에서 다음 행은 영수증 미첨부여도 인쇄/검증 통과합니다:
  • 유류비 (col-2) — 운행 거리 × 유가 자동 계산이라 영수증 없음
  • 통행료 (col-3 + 비고에 "통행료" 포함) — 자동 sync 행 + 수기 통행료 모두
  • 같은 행에 유류비/통행료만 함께 있어도 면제
그 외 컬럼(식비·교통비/택시·숙박·접대비 등)에 비용이 있으면 영수증 필수 유지. 해외 표는 카테고리 차이로 면제 규칙 적용 안 됨.

3. 식비 한도 · 1인당 정산

회사 규정에 따라 식비는 1인당 12,000원까지만 정산됩니다. 초과 금액은 자동으로 한도까지만 셀에 기록되며, 원본 금액은 비고에 [한도적용 원금 XX,XXX] 형태로 자동 기록됩니다.

한도 적용 예시

상황결과
1명 · 10,000원식비 10,000원 그대로
1명 · 15,000원식비 12,000원 + 비고 「[한도적용 원금 15,000] ...」
3명 · 30,000원식비 30,000원 (한도 36,000원 이내)
3명 · 40,000원식비 36,000원 + 비고 「[한도적용 원금 40,000] [3명] ...」

수기 입력 시 인원 반영

수동으로 항목을 추가한 경우(영수증 AI 업로드가 아닌 경우) 기본 인원은 1명 으로 처리됩니다.
여러 인원 회식이면 비고 칸에 [3명] 또는 [3인] 형태로 적어주세요. 자동으로 해당 행 인원수가 업데이트되고 한도가 재계산됩니다.
· 숙박도 동일 — [2박] 형태로 기입하면 반영
· 예: 식비 칸에 40,000 + 비고에 [3명] 탕화쿵푸마라탕 → 한도 36,000원(12,000×3) 적용되어 40,000 → 36,000으로 자동 캡

경고/상태 표시 색상 (전체)

  • 🔴 빨간색 배경 — 식비·숙박비 1인당 한도 초과 (자동 캡 적용됨)
  • 🟠 호박색 점선 — 출장일비 기간 내 식비 중복 청구 (이중 청구 위험)
  • 🟣 보라 테두리 — 접대비 상세 미입력 (더블클릭 → 모달 입력)
  • 🟢 초록 테두리 — 접대비 상세 입력 완료
💡 경고 사유 확인
· 경고 테두리/배경이 있는 셀에 마우스를 올리면 사유 툴팁 표시
· Shift + 클릭 → 사유 알림창으로 상세 표시
출장일비 기간 내 식비 경고 작동 방식
· 출장일비 기간(시작일~종료일) 내의 날짜에 식비가 입력되면 식비 셀 + 출장 기간 셀 양쪽 모두 호박색 경고로 표시됩니다.
· 예: 출장 기간 `2026-04-10 ~ 2026-04-12`, 개인경비 4월 11일 식비 → 양쪽 경고
· 해결: 해당 식비는 삭제하거나, 출장일비에서 해당 일자를 제외
⚠ 한도 설정은 회사 규정입니다. 변경이 필요한 경우 관리자에게 문의하세요. (서버 .env의 MEAL_LIMIT_PER_PERSON 설정)

3-1. 숙박비 한도 · 1인 1박 정산

국내 출장 숙박비는 1인 1박 70,000원까지만 정산됩니다. 초과 금액은 자동으로 한도까지만 셀에 기록되며, 원본 금액은 비고에 [한도적용 원금 XX,XXX] 형태로 자동 기록됩니다.

상황결과
1인 · 1박 · 60,000원숙박비 60,000원 그대로
1인 · 1박 · 90,000원숙박비 70,000원 + 비고 「[한도적용 원금 90,000] ...」
2인 · 2박 · 250,000원숙박비 250,000원 (한도 280,000원 이내)
2인 · 2박 · 320,000원숙박비 280,000원 (70,000×2×2) + 비고 「[한도적용 원금 320,000] ...」
박수(숙박일수) 자동 인식
· 영수증에 "2박", "2일" 또는 체크인/체크아웃 날짜가 있으면 AI가 자동 파싱
· 파일명에 출장숙박_2박.jpg 형태도 인식
· 인식 안 되면 기본 1박으로 처리 → 비고 칸 수동 수정
⚠ 한도 설정은 회사 규정입니다. 변경이 필요한 경우 관리자에게 문의하세요. (서버 .env의 LODGING_LIMIT_PER_NIGHT 설정)

3-2. 접대비 상세 입력

접대비는 감사·세무 증빙을 위해 거래처·참석자·목적 기록이 필수입니다. 시스템은 아래 흐름으로 동작합니다.

자동 감지

  • 파일명에 "접대" 포함 → 자동으로 접대비 카테고리로 분류
  • AI가 영수증 내용에서 접대성 표기(거래처, 회식) 인식 시 자동 분류

상세 입력 (📝 버튼)

  • 접대비 금액 셀 우측의 📝 버튼 클릭 → 상세 입력 모달 표시
  • 입력 항목: 하위구분(식사/음료/선물/경조사/회의비/기타), 거래처명, 내부 참석자 이름, 외부 참석자 이름/소속, 접대 목적
  • 저장 시 비고 칸에 구조화 형식으로 자동 삽입

상태 표시 색상

  • 🟣 보라 테두리 — 접대비 금액 입력됨, 상세 정보(거래처·목적) 미입력
  • 🟢 초록 테두리 — 상세 정보 입력 완료
  • 인쇄 시 테두리 자동 제거

비고 저장 형식 (예시)

[접대:음료] ULS / 지현태 / 조동환차장 / 접대

순서: 하위구분 / 거래처 / 내부참석자 / 외부참석자 / 목적

이 형식을 회계팀 검증 툴이 파싱해 자동 검증에 활용합니다.

4. 차량 유류비 정산

국내 개인경비 탭 상단의 🚗 차량 유류비 정산 섹션을 펼쳐 사용합니다. (기본 접힘 상태)

입력 순서

  1. 차종 · 차량번호 입력
  2. 유종 선택 (휘발유 8km/ℓ · 경유 7km/ℓ · LPG 5km/ℓ · 전기EV)
  3. 유종에 맞는 단가 입력 (유가 또는 전력량/전비/기본료)
  4. 항목 추가로 주행 건마다 일자 · 출발/도착지 · 거리(km) · 통행료 입력
  5. 정산 반영 버튼 클릭 → 자동 계산된 유류비와 통행료가 개인경비 표의 유류비/교통비 칼럼에 추가됩니다

계산 공식

  • 일반 연료: 유류비 = (주행거리 ÷ 연비) × 유가
  • 전기차: 유류비 = 기본료 + (주행거리 × 전력량단가 ÷ 복합전비)
  • 정산 반영 시 소수점 첫째자리에서 반올림하여 정수로 표 반영 (감사 일관성)

유가 설정값 가져오기

유가 입력 칸 옆의 🔄 자동 버튼 → 관리자가 ⚙️ 설정에 입력한 유가 값을 가져옵니다.
· 휘발유 / 경유 / LPG 모두 지원 (EV는 수동 입력)
· 설정에 값이 0 이거나 비어있으면 경고 표시 → 수기 입력
· 주간 고시가 바뀌면 관리자가 ⚙️ 설정에서 값만 갱신하면 전사 반영
유류비 내역이 없으면 인쇄 시 자동으로 유류비 페이지 생략됩니다.

5. 국내 출장일비 정산

국내 출장 시 1일 30,000원이 자동 계산됩니다 (관리자 설정으로 단가 변경 가능). 출장 건마다 항목 추가로 일수 · 기간 · 출장지를 입력하면 금액이 자동 산정됩니다.

출장 기간 입력 — 달력 선택

  • 기간 칸은 시작일 · 종료일 두 개의 날짜 선택기(달력) 로 구성되어 있습니다.
  • 각 칸 클릭 → 캘린더 팝업에서 날짜 선택
  • 수기 입력 불필요, 포맷 오류 원천 차단

이중 청구 방지 경고 (출장일비 ↔ 식비)

출장일비에는 식대가 이미 포함되어 있습니다. 같은 날짜에 개인경비 식비를 함께 청구하면 이중 청구가 되므로:
  • 출장 기간 내 날짜로 식비 입력 시 → 해당 식비 셀 + 출장 기간 셀 양쪽이 호박색 점선 경고
  • 출장 기간이 비어 있으면 경고 작동 안 함 → 반드시 기간 입력 먼저
  • 체크 시점: 식비 금액·월·일 입력, 출장 기간 수정 시마다 자동 재확인
출장일비 입력 내역(기간 · 일수 · 출장지)은 임시저장에 모두 포함됩니다.

6. 해외 개인경비 · 환율

해외 출장 영수증을 외화 그대로 입력하면 환율이 자동 적용되어 원화 환산액이 계산됩니다. 출장 기간 입력 후 평균 환율 일괄 연동 버튼으로 모든 행에 일괄 환율 적용도 가능합니다.

환율 조회 방식

  • 출처: 서울외국환중개(smbs.biz) 매매기준율 (단일 공신력 소스)
  • 단건 조회: 해외경비 행의 🔄 버튼 → 최근 영업일 매매기준율
  • 기간 평균: 출장 시작일/도착일 입력 후 평균 환율 일괄 연동 버튼 → 영업일 평균
  • 100단위 고시 통화 자동 환산: JPY · VND · IDR 은 ÷100 하여 1단위 기준으로 변환
  • CNY → CNH 자동 매핑: 서울외국환중개는 위안화를 CNH(역외)로 고시 → 사용자는 CNY 입력 그대로 OK

환율 표시 자릿수 (통화별 분기)

  • JPY · VND · IDR: 소수점 5번째 자리에서 반올림 → 4자리까지 표시 (값이 작아 정밀도 필요)
  • 그 외 통화 (USD/EUR/CNY 등): 소수점 3번째 자리에서 반올림 → 2자리까지 표시

원화 환산 공식

  • 해외경비: 원화 = 외화금액 × 환율 (행마다 자동 계산, 합계 셀에 표시)
  • 외화 금액 입력 시 모든 카테고리 칸이 소수점 2자리까지 자동 포맷팅

출장기간 인쇄 표시

해외 탭 상단에 입력한 📅 출장 기간은 인쇄 시 "해외 개인경비 정산" 헤더 옆에 자동 표시됩니다 (감사 추적용).

◈ 전도금(일비 제외) 차감

회사에서 미리 지급받은 출장 선지급금(전도금)이 있는 경우, 해외 개인경비 정산 섹션 헤더 우측의 ◈ 전도금(일비 제외) 입력 박스(항목 추가 버튼 좌측)에 KRW 금액을 입력하면 정산금액에서 자동 차감됩니다.

  • 위치: 해외 개인경비 정산 섹션 헤더 우측 — [항목 추가] 버튼 왼쪽의 흰 박스 안
  • 차감 범위: 해외경비 합계 에서만 차감 (※ 일비는 별도 정산되므로 제외)
  • 합계 공식 (rev. 5):
    • 표 footer 해외경비 합계 = (외화 행 원화환산 합) − 전도금 ← 차감 후 값을 직접 표시
    • 상단 인디고 카드 해외 합계 금액 = 해외경비 합계(차감 후) + 출장일비 합계
    • 두 값이 부제 (해외경비 합계 + 출장일비 합계) 와 정확히 매칭되어 검산이 직관적
  • 입력 즉시 footer & 상단 카드 동시 갱신, 부제 문구는 변경 없음
  • 임시저장·불러오기에 함께 영속화 (재로드 시 유지)
  • 인쇄 시에도 헤더에 그대로 노출 (감사 증빙)
전도금이 해외경비 합계보다 큰 경우 footer가 음수로 표시됩니다 (회사 환수액 의미). 0 입력 또는 빈 값 = 차감 없음.

⚠ 환율 미적용 행 시각 강조 (rev. 5 신규)

외화 금액은 입력했는데 환율이 비어 원화환산이 0인 행은 환율 칸 + 원화환산액 칸이 빨간 outline + 펄스(깜빡이는) 배경으로 강조됩니다. 검증 버튼을 누르지 않아도 입력 즉시 화면에서 식별 가능.

  • 발동 조건: 외화 금액 합 > 0 + 통화 입력됨 + 환율 = 0 (또는 비어 있음)
  • 해소: 환율 칸에 직접 입력하거나 [평균 환율 일괄 연동] 버튼 클릭
  • 인쇄: 강조는 자동 무력화되어 정상 인쇄 (검증 모달이 화면에서 안내 역할)
  • KRW(원화) 행은 환율 검사 대상에서 제외
서울외국환중개 사이트 장애 시 환율이 자동 입력되지 않을 수 있습니다. 이 경우 환율 칸에 수동 입력해 주세요.

6-1. 해외 출장일비 정산

해외 출장 시 회사 규정에 따른 지역·직급별 일비를 자동으로 적용합니다. 국내 출장일비와 동일한 테마(민트색)로 표시됩니다.

입력 컬럼

컬럼설명
번호행 추가/삭제 시 자동 재번호
총출장기간출장 시작일 ~ 종료일 (정보 보관용)
일비신청기간실제 일비 청구 기간 (총출장기간 범위 내로 제한). 입력 시 일수 자동 계산
국가/도시수기 입력 (예: 미국/뉴욕)
지역드롭다운 — 지역/직급 일비 기준 설정 모달에서 등록한 지역 목록 (콤보박스 펼치면 전체 설명 노출)

증빙: ※ 증빙 : 항공권 (정산서 출력 시 합계 행에 표기)

6-2. 지역/직급별 일비 기준 — 보기 / 편집

해외 출장일비의 자동 단가는 관리자 설정 모달 안의 일비 기준 섹션에서 관리합니다. 일반 사용자는 📋 일비 기준 보기 버튼으로 현재 설정만 확인할 수 있습니다.

📋 일비 기준 보기 (모든 사용자)

  • 해외 출장일비 정산 섹션 헤더의 📋 일비 기준 보기 버튼 클릭
  • 읽기 전용 뷰어 모달 → 지역 드롭다운으로 네비게이션 + 직급별 일비 표시
  • 편집 불가 — 수정이 필요하면 ⚙️ 관리자 설정으로 진입

⚙️ 관리자 설정 → 일비 기준 섹션 🔒 관리자

화면 우측 상단 ⚙️ 설정 버튼 → 비밀번호 입력 → "🌍 지역/직급별 일비 기준" 섹션에서 다음 작업 가능:

  • 지역 드롭다운 — 기존 지역 선택해 단가 수정
  • ＋ 추가 버튼 — 이름·설명 prompt → 새 지역 생성
  • 🗑 삭제 버튼 — 현재 지역 삭제 (최소 1개 유지)
  • 지역명 / 설명 / 7개 직급(임원·부장·차장·과장·대리·주임·사원) 원화 단가 입력

저장 흐름

  1. 관리자 모달에서 일비 기준을 추가/수정/삭제
  2. 모달 하단 비밀번호 입력 후 저장 클릭
  3. 서버 비밀번호 검증 → 통과 시 한도/유가/일비 모두 동시 저장
  4. 모든 출장일비 행의 단가가 자동 일괄 갱신
  5. 취소 또는 모달 닫기 → 변경사항 자동 롤백 (실수 방지)
저장 위치 (rev. 5 변경): 일비 기준 / 한도 / 유가 모두 서버 settings.json 에 저장됩니다 — 전사 공통 적용. 한 명의 관리자가 [저장]을 누르면 모든 사용자가 같은 기준을 보게 됩니다.

7. 임시저장 · 불러오기

상단 우측 임시저장 / 불러오기 버튼으로 현재 입력 내역을 브라우저에 저장/복원할 수 있습니다.

저장 범위

  • 작성자 정보(성명 · 부서 · 차수 · 작성일)
  • 차량 정보(차종 · 차량번호) 및 유종 설정
  • 유류비 주행 로그 전체
  • 출장일비 행 전체
  • 개인경비 표 각 행(월/일/금액/비고)
  • 해외 출장 시작일/도착일 및 해외경비 표
브라우저 localStorage에 저장되므로 다른 PC·브라우저·시크릿 모드에서는 볼 수 없습니다. 영구 보존은 인쇄/PDF로 저장해 주세요.

7-1. 상단 툴바 기능

탭 전환 영역 우측에 좌→우 순으로 다음 버튼이 있습니다: 개인정보 삭제 · 임시저장 · 불러오기 · 증빙PDF · 검증 · ⚙️ 설정

🩷 증빙 PDF (미리보기 + 다운로드)

  • 모달 하단의 📥 다운로드 버튼 → 병합 PDF 파일 저장

  • 같은 파일을 두 번 업로드해도 PDF에는 한 번만 포함 (중복 자동 제거)
  • 다른 탭의 영수증은 그 탭으로 전환 후 별도 추출
  • 브라우저 메모리에 임시 보관 — 새로고침 시 사라짐. 업로드 후 즉시 미리보기·다운로드 권장

⚙️ 관리자 설정 (한도 · 정책 · 일비)

  • ⚙️ 설정 버튼 → 비밀번호 입력 후 회사 규정 한도 변경 (서버 재시작 불필요, 즉시 반영)
  • 설정 항목
    • 국내 식비 한도 (1인당, 원)
    • 국내 숙박비 한도 (1인 1박, 원)
    • 해외 식비 한도 (1인당, 원화환산 기준) — 0 입력 시 미적용
    • 해외 숙박비 한도 (1인 1박, 원화환산 기준) — 0 입력 시 미적용
    • 해외 한도 초과 시 동작 (경고만 / 자동 캡)
    • 국내 출장일비 (1일, 원)
    • ⛽ 유가 (휘발유 / 경유 / LPG, 원/L) — 주간 고시 기준 수동 입력

  • 비밀번호: 기본 admin (서버 .env의 ADMIN_PASSWORD로 변경 가능)
  • 한도/유가/일비 모두 한 번의 [저장]으로 일괄 반영
  • 한도·유가·일비 기준 모두 서버 settings.json 저장 (전사 공통)
  • 모달 [취소] 시 일비 기준 변경사항도 자동 롤백 (실수 방지)

🟡 제출 전 규정 검증

  • 검증 버튼 → 제출 전 규정·증빙 항목 자동 체크 (현재 활성 탭 무관 — 국내·해외 모두 검사)
  • 결과 분류
    • 🔴 오류 — 반드시 수정 (작성자 누락 / 합계 불일치 / 이중 청구 / 접대비 미입력 / 수기 행 영수증 미첨부 / 해외 출장기간 미입력 / 평균환율 미적용·불일치 등)
    • 🟡 경고 — 확인 권장 (월/일 누락 / 비고 누락 / 출장지 누락 등)
    • 🔵 참고 — 식비 한도 자동 캡 적용 / 3만원 이상 접대비 적격증빙 체크 등
  • 모든 항목이 정상이면 "✅ 검증 통과" 녹색 박스 표시
해외 환율 검증 (인쇄 차단 조건과 동일)
1. 출장기간(시작일·종료일) 입력 여부
2. 현재 출장기간으로 [평균 환율 일괄 연동] 버튼이 클릭되었는지
3. 모든 외화 행의 환율이 평균환율과 정확히 일치하는지 (수동 환율, 평균 적용 후 추가된 행 모두 검출)
위 3가지를 통과해야 인쇄 가능합니다.
입력 단계 시각 강조 (rev. 5)
검증 버튼을 누르지 않아도 외화 금액 + 통화는 있는데 환율이 비어 있는 행은 환율 칸 / 원화환산액 칸이 빨간 outline + 펄스 배경으로 즉시 강조됩니다. 검증 모달 🔴 오류 항목과 동일한 행을 표면화.

8. 정산서 인쇄 · PDF

상단의 정산서 인쇄 버튼을 누르면 인쇄 범위 선택 모달이 먼저 뜹니다. 여기서 출력 범위를 고른 뒤 브라우저 인쇄 대화상자에서 프린터 출력 또는 "PDF로 저장"을 선택합니다.

인쇄 범위 선택 (3가지 옵션)

옵션출력 내용
🇰🇷 국내국내 정산서 + 국내 영수증 증빙 PDF만
✈️ 해외해외 정산서 + 해외 영수증 증빙 PDF만
📑 전체국내 정산서 → 국내 증빙 → 해외 정산서 → 해외 증빙 (이 순서)
선택한 범위에 해당하지 않는 검증(예: 국내만 인쇄 시 해외 환율 검증)은 자동으로 건너뜁니다. 활성 탭에 정산 금액이 0인 옵션을 선택하면 안내 메시지가 뜹니다.

자동 최적화 동작

  • 정산 금액이 0원인 탭(국내 또는 해외)은 인쇄 제외
  • 유류비 내역이 비어 있으면 유류비 페이지 생략
  • 국내 출장일비 합계가 0이면 출장일비 섹션 인쇄 제외 (해외도 동일)
  • 비어 있는 입력 행(월/일/금액 모두 공란)은 인쇄 시 자동 숨김
  • 식비 한도 초과 경고(빨간색 배경)는 인쇄 시 제거
  • 배경색·볼드·컬러 헤더는 인쇄용 중립 톤으로 자동 변환
  • 해외 탭 인쇄 시 헤더 옆에 📅 출장기간 자동 노출 (작은 글씨로 헤더와 균형)
  • 수기 입력 행 식별 배지 유지 — 비고 셀 우측에 흰 바탕 검정 테두리 "수기" 배지 (AI 행은 표시 안 됨)
  • 출장일비 표 인쇄 글자 크기 일관화 (본문 10.5px) + 합계 행 강조 (라벨 13px / 총액 17px) — 표 내용 전체 약 10% 확대 (화면도 동일 비율)
  • 증빙 영수증 자동 첨부 — 정산서 다음에 해당 탭 영수증이 한 장씩 한 페이지에 맞춰 자동 인쇄 (object-fit: contain · PDF.js 동적 로드)
  • 전체 인쇄 시 순서: 국내 정산서 → 국내 증빙 → 해외 정산서 → 해외 증빙

인쇄 차단 조건 (제출 전 자동 검증)

아래 조건 중 하나라도 해당되면 인쇄가 차단되고 안내 메시지가 표시됩니다:
  1. 수기 입력 행 영수증 미첨부 — 회색 📎+ 버튼으로 영수증을 먼저 첨부 (단 유류비 / 통행료 행은 면제)
  2. 해외 출장 기간 미입력 — 시작일·종료일 모두 입력 필요
  3. 해외 [평균 환율 일괄 연동] 미적용 — 현재 출장기간으로 버튼 클릭 필수
  4. 해외 외화 행 환율이 평균환율과 불일치 — 평균 적용 후 추가된 행이나 수동 환율을 사용한 행이 있을 때 (다시 [평균 환율 일괄 연동] 클릭하면 해소)
원화(KRW) 행은 환율 검사에서 제외됩니다.

증빙 PDF 자동 첨부 (인쇄 시)

  • 정산서 인쇄 버튼 클릭 시 현재 탭 영수증을 자동 병합 → PDF.js로 페이지마다 이미지 렌더 → 정산서 다음에 한 장씩 첨부
  • 이미지(PNG/JPG) 영수증 + PDF 영수증 모두 지원
  • 첫 인쇄 시 PDF.js를 외부 CDN 에서 로드 (1~2초 추가)
  • 영수증이 없거나 PDF.js 로드 실패 시 정산서만 정상 인쇄 (graceful fallback)
인쇄 대화상자의 배율은 100%로 두는 것을 권장합니다. 용지는 A4 세로/가로 자동으로 처리됩니다.

9. 자주 묻는 질문

Q. 영수증을 업로드했는데 결과가 표시 안 됩니다.

A. AI 모델이 영수증을 인식하지 못한 경우입니다. 화면 하단의 ❌ 분석 실패 항목 목록을 확인하고, 해당 영수증은 항목 추가로 수동 입력해 주세요.

Q. 주차비 영수증이 교통비 칸에 들어갔어요.

A. 최신 버전에서 수정되었습니다. 서버가 구버전이면 관리자에게 업데이트 요청하세요.

Q. 식비 한도가 회사 규정과 다릅니다.

A. 기본 12,000원/1인 입니다. 변경은 서버 관리자에게 문의 (MEAL_LIMIT_PER_PERSON 환경변수).

Q. 환율이 자동으로 안 불러와집니다.

A. 서울외국환중개 사이트 접속 장애일 수 있습니다. 잠시 후 재시도하거나, 환율을 수동 입력해 주세요.

Q. 임시저장한 데이터가 사라졌습니다.

A. 브라우저 캐시 삭제 · 시크릿 모드 종료 · 다른 PC 로그인 시 사라집니다. 중요한 내역은 PDF로 저장해 두세요.

Q. 인쇄하면 유류비 섹션이 빈 1페이지로 나옵니다.

A. 최신 버전에서는 유류비 내역이 없으면 유류비 페이지를 자동 생략합니다. 구버전이면 업데이트 필요.

Q. 영수증을 삭제했는데 증빙PDF에 그대로 들어갑니다.

A. 최신 버전에서 수정되었습니다. 증빙PDF 버튼 클릭 시점의 표 상태를 기준으로 PDF가 만들어집니다 — 삭제한 행의 영수증은 자동 제외됩니다.

Q. 영수증을 다시 보고 싶은데 어디서 보나요?

A. AI로 추가된 행 우측의 📎 페이퍼클립 아이콘 클릭 → 모달에서 원본 즉시 표시. 단, 페이지 새로고침 후엔 메모리에서 사라지므로 미리보기는 동작하지 않습니다 (영수증을 다시 업로드하면 복원).

Q. 2025년 영수증인데 표에는 2026년으로 들어갑니다.

A. AI가 영수증의 날짜를 정확히 인식하지 못한 경우 작성일자 연도로 자동 보정됩니다. 연 칸을 클릭해 직접 수정 가능 — 수정 즉시 자동 정렬됩니다.

Q. 일비 기준 단가를 바꿨는데 행에 반영이 안 됩니다.

A. ⚙️ 관리자 설정 모달에서 일비 기준 섹션 수정 후 [저장] 버튼을 눌러야 적용됩니다. 비밀번호 검증 통과 시 이미 입력된 출장일비 행의 단가도 자동 일괄 갱신됩니다. (취소·X 클릭 시 변경사항은 폐기)

Q. AI가 영수증 인식을 실패한 경우 어떻게 해야 하나요?

A. (1) 항목 추가로 행을 만든 뒤, (2) 행 우측의 📎+ 회색 페이퍼클립 버튼 클릭 → 영수증 파일 첨부, (3) 자동으로 파란 📎 미리보기로 전환됩니다. 첨부한 영수증은 증빙PDF 병합에도 자동 포함됩니다.

Q. 인쇄가 갑자기 안 됩니다 — "환율이 적용되지 않은 행" 메시지가 떠요.

A. 해외 탭에서 외화로 입력된 행 중 환율이 비어있는 행이 있으면 인쇄가 차단됩니다. 해외 탭의 [평균 환율 일괄 연동] 버튼으로 출장 기간 평균 환율을 일괄 적용하거나, 각 행의 🔄 버튼으로 단건 적용 후 다시 인쇄하세요. (원화 행은 차단 대상 아님)

Q. 인쇄물에서 어떤 행이 수기 입력인지 어떻게 구분하나요?

A. 수기 입력 행은 인쇄 시 비고 셀 우측에 작은 흰 바탕 검정 테두리 "수기" 배지가 남습니다 (영수증 아이콘이 있던 위치). AI 분석 영수증 행은 인쇄에서 배지가 숨겨집니다.

Q. 증빙PDF 버튼을 눌렀더니 화면에 PDF가 표시됩니다. 다운로드는요?

A. 새 미리보기 모달에서 인라인 PDF 뷰어로 먼저 확인하고, 모달 하단의 📥 다운로드 버튼을 누르면 파일이 저장됩니다. 다른 탭(국내↔해외)의 영수증을 받으려면 그 탭으로 전환 후 다시 클릭하세요.

Q. 정산서 인쇄에 영수증도 같이 인쇄되게 하려면?

A. 별도 동작 불필요 — 인쇄 버튼을 누르면 인쇄 범위 선택 모달에서 국내/해외/전체 중 선택 후 정산서 다음에 해당 탭 영수증이 한 페이지씩 자동 첨부되어 출력됩니다. 첫 인쇄 시 PDF.js 로드로 1~2초 더 걸릴 수 있습니다.

Q. 해외 출장 전도금(선지급금)은 어디에 입력하나요?

A. 해외 탭 → 해외 개인경비 정산 섹션 헤더 우측, [항목 추가] 버튼 왼쪽의 ◈ 전도금(일비 제외) 입력 박스에 KRW 금액을 입력하세요. 입력 즉시 해외경비 합계에서 차감되어 정산금액이 자동 갱신됩니다 (출장일비는 차감 대상 아님). 입력값은 임시저장에도 함께 보관됩니다.

Q. 인쇄할 때 국내 또는 해외만 출력하고 싶어요.

A. 정산서 인쇄 버튼을 누르면 뜨는 모달에서 🇰🇷 국내 또는 ✈️ 해외를 선택하면 해당 탭의 정산서와 증빙 PDF만 출력됩니다. 둘 다 함께 출력하려면 📑 전체를 선택하세요.

Q. 수기로 행을 추가했는데 인쇄가 안 됩니다.

A. 수기 행에는 영수증 첨부가 필수입니다. 행 우측의 회색 📎+ 버튼을 클릭해 영수증을 첨부한 뒤 다시 인쇄하세요. 검증 버튼으로 미리 점검 가능. 예외: 국내 표의 유류비(col-2) / 통행료(col-3 + 비고 "통행료") 행은 영수증 면제 (rev. 5).

Q. 검증 버튼을 눌렀는데 해외 환율 누락이 안 잡힙니다.

A. 최신 버전(rev. 5)에서는 활성 탭 무관하게 해외 항목을 항상 검사하고, 환율이 빠진 행은 입력 즉시 빨간 outline + 펄스 배경으로 화면에 표시됩니다. 출장기간 미입력 / [평균 환율 일괄 연동] 미적용 / 환율이 평균과 다른 행 모두 🔴 오류로 표시됩니다.

Q. 해외경비 합계 (footer) 와 해외 합계 금액 (상단 카드) 숫자가 다릅니다.

A. rev. 5 부터는 두 값이 부제 식과 정확히 매칭됩니다.
· footer 해외경비 합계 = (외화 원화환산 합) − 전도금
· 상단 카드 해외 합계 금액 = footer + 출장일비 합계
전도금이 0이고 일비도 0이면 두 값이 같아집니다. 일비가 있거나 전도금 차감이 있으면 그 차이만큼 다르게 표시되며, 그게 정상입니다.

Q. 공용 PC에서 사용 후 내 정보를 지우려면?

A. 툴바 맨 좌측의 👤 개인정보 삭제 버튼 → 확인 후 즉시 삭제. 다음 사용자가 빈 화면으로 시작하게 됩니다.
