1. 시작하기

본 시스템은 회사 개인경비 영수증을 업로드하면 AI가 자동으로 금액·날짜·카테고리를 인식하여 정산서를 작성해 주는 도구입니다.

기본 흐름

  1. 상단 탭에서 국내 개인경비 또는 해외 개인경비 선택 🚧 해외 개발 중
  2. 작성자 정보(성명 · 소속 부서 · 정산 차수 · 작성 일자) 입력
  3. 영수증 업로드(AI 분석) 또는 항목 추가 버튼으로 수동 입력
  4. 필요 시 차량 유류비 · 출장일비 입력
  5. 정산서 인쇄 버튼으로 출력 또는 PDF 저장

2. 영수증 AI 자동 분석

업로드 방법

  • 업로드 영역을 클릭하여 파일 선택 (여러 장 가능)
  • 파일을 영역으로 드래그&드롭
  • 클립보드에 캡처한 이미지를 Ctrl + V 붙여넣기
  • 지원 포맷: 이미지(PNG · JPG 등) · PDF
✅ 기존 입력 내역은 유지됩니다. 영수증을 추가로 업로드해도 이미 입력한 행은 그대로 남고, 새 영수증만 하단에 추가됩니다. (중복 업로드 시 같은 영수증은 자동 무시)
⚠ 한 파일당 영수증 1장만 업로드하세요.
여러 영수증이 한 파일 안에 같이 스캔·촬영되어 있으면 AI가 가장 눈에 띄는 한 건만 인식하거나, 금액·항목을 뒤섞어 잘못 판독할 수 있습니다.
· PDF 여러 페이지 = 각 페이지가 별도 영수증으로 처리됨 (OK)
· 이미지 한 장에 영수증 2장 붙여넣기 = 권장하지 않음 (1장당 1파일 원칙)

AI가 자동 인식하는 항목

항목설명
월/일영수증의 결제일
금액합계 금액(마이너스 기호는 절대값으로 변환)
카테고리식비·유류비·교통비·주차비·숙박비·접대비·소모품·운반비·기타
업체명비고 칸에 자동 기입
인원수영수증에 "3명/3인/3인분" 표기가 있으면 자동 인식
인원수 인식 팁
· 영수증에 손글씨·스탬프로 "3명" "4인" 등이 적혀 있으면 AI가 읽어냅니다.
· 파일명에 점심_3명.jpg 처럼 표기해도 인식됩니다.
· AI가 못 읽은 경우 기본 1명으로 처리되며, 비고 칸 수정으로 대응 가능합니다.

3. 식비 한도 · 1인당 정산

회사 규정에 따라 식비는 1인당 12,000원까지만 정산됩니다. 초과 금액은 자동으로 한도까지만 셀에 기록되며, 원본 금액은 비고에 [한도적용 원금 XX,XXX] 형태로 자동 기록됩니다.

한도 적용 예시

상황결과
1명 · 10,000원식비 10,000원 그대로
1명 · 15,000원식비 12,000원 + 비고 「[한도적용 원금 15,000] ...」
3명 · 30,000원식비 30,000원 (한도 36,000원 이내)
3명 · 40,000원식비 36,000원 + 비고 「[한도적용 원금 40,000] [3명] ...」

수기 입력 시 인원 반영

수동으로 항목을 추가한 경우(영수증 AI 업로드가 아닌 경우) 기본 인원은 1명 으로 처리됩니다.
여러 인원 회식이면 비고 칸에 [3명] 또는 [3인] 형태로 적어주세요. 자동으로 해당 행 인원수가 업데이트되고 한도가 재계산됩니다.
· 숙박도 동일 — [2박] 형태로 기입하면 반영
· 예: 식비 칸에 40,000 + 비고에 [3명] 탕화쿵푸마라탕 → 한도 36,000원(12,000×3) 적용되어 40,000 → 36,000으로 자동 캡

경고 표시 2종

  • 🔴 빨간색 배경 — 1인당 한도 초과 (자동 캡 적용됨)
  • 🟠 호박색 점선 — 출장일비 기간 내 식비 중복 청구 (이중 청구 위험). 출장일비에는 식대가 포함돼 있으므로 같은 날짜 식비는 별도 청구 불가
출장일비 기간 내 식비 경고 작동 방식
· 출장일비 기간(시작일~종료일) 내의 날짜에 식비가 입력되면 식비 셀 + 출장 기간 셀 양쪽 모두 호박색 경고로 표시됩니다.
· 예: 출장 기간 `2026-04-10 ~ 2026-04-12`, 개인경비 4월 11일 식비 → 양쪽 경고
· 해결: 해당 식비는 삭제하거나, 출장일비에서 해당 일자를 제외
⚠ 한도 설정은 회사 규정입니다. 변경이 필요한 경우 관리자에게 문의하세요. (서버 .env의 MEAL_LIMIT_PER_PERSON 설정)

3-1. 숙박비 한도 · 1인 1박 정산

국내 출장 숙박비는 1인 1박 70,000원까지만 정산됩니다. 초과 금액은 자동으로 한도까지만 셀에 기록되며, 원본 금액은 비고에 [한도적용 원금 XX,XXX] 형태로 자동 기록됩니다.

상황결과
1인 · 1박 · 60,000원숙박비 60,000원 그대로
1인 · 1박 · 90,000원숙박비 70,000원 + 비고 「[한도적용 원금 90,000] ...」
2인 · 2박 · 250,000원숙박비 250,000원 (한도 280,000원 이내)
2인 · 2박 · 320,000원숙박비 280,000원 (70,000×2×2) + 비고 「[한도적용 원금 320,000] ...」
박수(숙박일수) 자동 인식
· 영수증에 "2박", "2일" 또는 체크인/체크아웃 날짜가 있으면 AI가 자동 파싱
· 파일명에 출장숙박_2박.jpg 형태도 인식
· 인식 안 되면 기본 1박으로 처리 → 비고 칸 수동 수정
⚠ 한도 설정은 회사 규정입니다. 변경이 필요한 경우 관리자에게 문의하세요. (서버 .env의 LODGING_LIMIT_PER_NIGHT 설정)

4. 차량 유류비 정산

국내 개인경비 탭 상단의 🚗 차량 유류비 정산 섹션을 펼쳐 사용합니다. (기본 접힘 상태)

입력 순서

  1. 차종 · 차량번호 입력
  2. 유종 선택 (휘발유 8km/ℓ · 경유 7km/ℓ · LPG 5km/ℓ · 전기EV)
  3. 유종에 맞는 단가 입력 (유가 또는 전력량/전비/기본료)
  4. 항목 추가로 주행 건마다 일자 · 출발/도착지 · 거리(km) · 통행료 입력
  5. 정산 반영 버튼 클릭 → 자동 계산된 유류비와 통행료가 개인경비 표의 유류비/교통비 칼럼에 추가됩니다

계산 공식

  • 일반 연료: 유류비 = (주행거리 ÷ 연비) × 유가
  • 전기차: 유류비 = 기본료 + (주행거리 × 전력량단가 ÷ 복합전비)
유류비 내역이 없으면 인쇄 시 자동으로 유류비 페이지 생략됩니다.

5. 출장일비 정산

국내 출장 시 1일 30,000원이 자동 계산됩니다. 출장 건마다 항목 추가로 일수 · 기간 · 출장지를 입력하면 금액이 자동 산정됩니다.

출장 기간 입력 — 달력 선택

  • 기간 칸은 시작일 · 종료일 두 개의 날짜 선택기(달력) 로 구성되어 있습니다.
  • 각 칸 클릭 → 캘린더 팝업에서 날짜 선택
  • 수기 입력 불필요, 포맷 오류 원천 차단

이중 청구 방지 경고 (출장일비 ↔ 식비)

출장일비에는 식대가 이미 포함되어 있습니다. 같은 날짜에 개인경비 식비를 함께 청구하면 이중 청구가 되므로:
  • 출장 기간 내 날짜로 식비 입력 시 → 해당 식비 셀 + 출장 기간 셀 양쪽이 호박색 점선 경고
  • 출장 기간이 비어 있으면 경고 작동 안 함 → 반드시 기간 입력 먼저
  • 체크 시점: 식비 금액·월·일 입력, 출장 기간 수정 시마다 자동 재확인
출장일비 입력 내역(기간 · 일수 · 출장지)은 임시저장에 모두 포함됩니다.

6. 해외 개인경비 · 환율 🚧 개발 중

해외 탭은 현재 개발 중이며 일부 기능이 동작합니다.

환율 조회 방식

  • 출처: 서울외국환중개(smbs.biz) 매매기준율 (단일 공신력 소스)
  • 단건 조회: 해외경비 행의 🔄 버튼 → 최근 영업일 매매기준율
  • 기간 평균: 출장 시작일/도착일 입력 후 평균 환율 일괄 연동 버튼 → 영업일 평균
  • JPY 특수 처리: 서울외국환중개는 100엔 기준 고시 → 자동으로 ÷100 하여 1엔 기준 환산

원화 환산 공식

  • 일반 해외경비: 원화 = 외화금액 × 환율
  • 해외 출장일비: 원화 = 일수 × 외화단가 × 환율
서울외국환중개 사이트 장애 시 환율이 자동 입력되지 않을 수 있습니다. 이 경우 환율 칸에 수동 입력해 주세요.

7. 임시저장 · 불러오기

상단 우측 임시저장 / 불러오기 버튼으로 현재 입력 내역을 브라우저에 저장/복원할 수 있습니다.

저장 범위

  • 작성자 정보(성명 · 부서 · 차수 · 작성일)
  • 차량 정보(차종 · 차량번호) 및 유종 설정
  • 유류비 주행 로그 전체
  • 출장일비 행 전체
  • 개인경비 표 각 행(월/일/금액/비고)
  • 해외 출장 시작일/도착일 및 해외경비 표
브라우저 localStorage에 저장되므로 다른 PC·브라우저·시크릿 모드에서는 볼 수 없습니다. 영구 보존은 인쇄/PDF로 저장해 주세요.

8. 정산서 인쇄 · PDF

상단의 정산서 인쇄 버튼 → 브라우저 인쇄 대화상자에서 프린터 출력 또는 "PDF로 저장" 선택.

자동 최적화 동작

  • 정산 금액이 0원인 탭(국내 또는 해외)은 인쇄 제외
  • 유류비 내역이 비어 있으면 유류비 페이지 생략
  • 비어 있는 입력 행(월/일/금액 모두 공란)은 인쇄 시 자동 숨김
  • 식비 한도 초과 경고(빨간색 배경)는 인쇄 시 제거
  • 배경색·볼드·컬러 헤더는 인쇄용 중립 톤으로 자동 변환
인쇄 대화상자의 배율은 100%로 두는 것을 권장합니다. 용지는 A4 세로/가로 자동으로 처리됩니다.

9. 자주 묻는 질문

Q. 영수증을 업로드했는데 결과가 표시 안 됩니다.

A. AI 모델이 영수증을 인식하지 못한 경우입니다. 화면 하단의 ❌ 분석 실패 항목 목록을 확인하고, 해당 영수증은 항목 추가로 수동 입력해 주세요.

Q. 주차비 영수증이 교통비 칸에 들어갔어요.

A. 최신 버전에서 수정되었습니다. 서버가 구버전이면 관리자에게 업데이트 요청하세요.

Q. 식비 한도가 회사 규정과 다릅니다.

A. 기본 12,000원/1인 입니다. 변경은 서버 관리자에게 문의 (MEAL_LIMIT_PER_PERSON 환경변수).

Q. 환율이 자동으로 안 불러와집니다.

A. 서울외국환중개 사이트 접속 장애일 수 있습니다. 잠시 후 재시도하거나, 환율을 수동 입력해 주세요.

Q. 임시저장한 데이터가 사라졌습니다.

A. 브라우저 캐시 삭제 · 시크릿 모드 종료 · 다른 PC 로그인 시 사라집니다. 중요한 내역은 PDF로 저장해 두세요.

Q. 인쇄하면 유류비 섹션이 빈 1페이지로 나옵니다.

A. 최신 버전에서는 유류비 내역이 없으면 유류비 페이지를 자동 생략합니다. 구버전이면 업데이트 필요.
